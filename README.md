**Mysql**
- Go to folder -> src/Mysql/ where you will get the Script for the creation of the Database and the Stored procedures

**NodeJs**


```sh
    npm install
```


**ENV**  
- APP_PORT 
- APP_KEY_JWT
- CLOUD_NAME
- CLOUD_API_KEY
- CLOUD_API_SEC
